HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML provides the structure and framework for our webpage.

CSS

  • CSS is the language we use to style an HTML document.
  • CSS describes how HTML elements should be displayed.
  • External style sheets are the most common way to style a webpage.
  • External style sheets allow developers to store theie CSS rules in a separate file.

Git

  • git status allow us to check what branch we are on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • Javascript enables us to add functionality to webpages.
